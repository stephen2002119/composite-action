# composite-action

## Ada Lovelace ASCII Art

This repository includes a bash shell script that displays ASCII art of Ada Lovelace, the first computer programmer.

### Usage

The script is executable. Simply run:

```bash
./ada_lovelace.sh
```

If needed, make it executable first:

```bash
chmod +x ada_lovelace.sh
./ada_lovelace.sh
```

The script displays ASCII art depicting Ada Lovelace along with inspirational quotes from her work.
